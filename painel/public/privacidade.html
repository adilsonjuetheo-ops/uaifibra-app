UaiFibra Telecom

Política de Privacidade

Última atualização: 17 de junho de 2026

1 Sobre esta Política

Esta Política de Privacidade descreve como a UaiFibra Telecom coleta, usa e protege as informações dos clientes que utilizam o aplicativo móvel UaiFibra (disponível para Android e iOS).

Ao utilizar o aplicativo, você concorda com as práticas descritas neste documento, em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD — Lei nº 13.709/2018).

2 Dados Coletados

O aplicativo coleta apenas os dados necessários para o funcionamento dos serviços:

  • CPF: utilizado como identificador de autenticação junto ao sistema do provedor.
  • Senha: transmitida de forma segura ao sistema IXC Soft do provedor; não é armazenada em texto simples no dispositivo.
  • Cidade: coletada a partir do cadastro do contrato, usada para filtrar notificações e avisos por região.
  • Token de notificação push: identificador anônimo gerado pelo serviço Expo/FCM/APNs para envio de notificações. Pode ser coletado mesmo sem login.
  • Dados do contrato: nome, situação da conexão, faturas e vencimentos — lidos diretamente do sistema do provedor no momento da consulta.
O aplicativo não coleta localização geográfica, contatos, câmera, microfone, dados bancários nem qualquer informação além das listadas acima.

3 Finalidade do Uso dos Dados

  • Autenticação: verificar a identidade do cliente para acesso à conta.
  • Consulta de faturas: exibir segunda via de boletos e situação do contrato.
  • Notificações push: enviar avisos de vencimento, manutenções e comunicados do provedor.
  • Avisos regionais: exibir informações relevantes para a cidade do cliente.
  • Desbloqueio temporário: solicitar reativação da conexão por até 24 horas em caso de atraso.

4 Armazenamento e Segurança

As credenciais de acesso (CPF e senha) são armazenadas localmente no dispositivo utilizando SecureStore (armazenamento criptografado do sistema operacional), exclusivamente para permitir o desbloqueio biométrico sem redigitar a senha.

Os tokens de notificação push são armazenados em banco de dados seguro (PostgreSQL) hospedado na infraestrutura do provedor, associados opcionalmente ao identificador do contrato e à cidade.

Nenhum dado pessoal é vendido, alugado ou compartilhado com terceiros para fins comerciais.

5 Compartilhamento de Dados

Os dados são compartilhados apenas com:

  • IXC Soft (sistema de gestão do provedor): para autenticação e consulta de dados contratuais. As informações trafegam de forma segura entre o aplicativo e o sistema do provedor.
  • Expo / Google Firebase / Apple APNs: para entrega de notificações push. Apenas o token anônimo do dispositivo é compartilhado com esses serviços.

Não há compartilhamento com parceiros de publicidade ou análise de dados.

6 Retenção de Dados

Os tokens de notificação push são mantidos enquanto o cliente utilizar o aplicativo. Ao desinstalar o aplicativo, o token deixa de ser válido automaticamente.

As credenciais armazenadas localmente no dispositivo são apagadas ao realizar logout ou ao desinstalar o aplicativo.

7 Seus Direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados.
  • Acessar os dados que possuímos sobre você.
  • Solicitar a correção de dados incompletos ou incorretos.
  • Solicitar a exclusão dos seus dados de notificação push.
  • Revogar o consentimento para recebimento de notificações (nas configurações do dispositivo).

Para exercer esses direitos, entre em contato pelo e-mail ou WhatsApp indicados abaixo.

8 Contato

Em caso de dúvidas sobre esta política ou sobre o tratamento dos seus dados, entre em contato com a UaiFibra Telecom:

  • WhatsApp: disponível no aplicativo, na seção "Suporte".
  • Atendimento presencial: nas unidades da UaiFibra Telecom.

9 Alterações nesta Política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas via notificação push ou aviso dentro do aplicativo. A data de "última atualização" no topo desta página sempre refletirá a versão vigente.
